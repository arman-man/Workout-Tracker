# Workout-Tracker
Designed and implemented a full-stack web application that allows users to track their workouts over time. 

Front-end: React.js, Node.js, HTML, CSS.
Built the front-end using React.js to provide a user-friendly UI to perform CRUD operations.

Back-end: Express.js, Node.js, MongoDB, REST API.
Implemented the back-end using Node.js to serve and modify data using route endpoints. Utilized MongoDB to store persistent workout data.
