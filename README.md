# Workout-Tracker
Designed and implemented a full-stack web application that allows users to track their workouts over time. 

Front-end: React.js, HTML, CSS.

Back-end: Node.js, Express.js, REST API, MongoDB.

## Installation
```npm install``` in the root directory of the REST-API and React-UI folders

## Running
```npm start``` in the root directory of the REST-API and React-UI folders
